# resume_kata

You can generate pdf version of your markdown resume.

# How to

Required Docker

## Create you resume

```sh
$ git clone git@github.com:garigari-kun/resume_kata.git
$ cd resume_kata
$ vi docs/README.md # Edit your resume!
```

## Build & Generate PDF version of your resume
```sh
$ make build
$ make build_resume
$ open output.pdf
```
